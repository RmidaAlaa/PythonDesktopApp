Pour bien commencer

Cet aperçu reprend la structure d’un document « Pour bien commencer » pour valider la mise en page et la navigation.

Ce qu’il vous faut

  • PC Windows 10/11, application installée, câble USB, appareils alimentés.
  • Optionnel : Internet, identifiants SMTP, compte OneDrive.

Configuration requise

  • Windows 10/11 (x64), ~200 Mo disque, permissions d’accès USB.
